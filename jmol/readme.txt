The most recent double-precision Jmol version that implements the project CIF format, as found at 
https://github.com/BobHanson/Jmol-SwingJS/tree/master/dist as JmolD.jar

JmolDataD.jar is the double-precision version of Jmol 
that is "headless" and best used for back-office batch operation.
See the script in test/spt, for example.
